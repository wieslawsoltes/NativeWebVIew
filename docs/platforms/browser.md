# Browser

## Backend

- Package: `NativeWebView.Platform.Browser`
- Platform enum: `NativeWebViewPlatform.Browser`

## Supported areas

- Embedded view
- Authentication broker
- New window interception
- Environment/controller options
- Native handles
- Cookie and command manager

## Unsupported in this phase

- Dialog backend
- Desktop windowing features

## Registration

```csharp
factory.UseNativeWebViewBrowser();
```
