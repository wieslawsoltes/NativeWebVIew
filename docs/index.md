# NativeWebView

NativeWebView is a native-webview-first control stack for Avalonia.

## Included surfaces

- Embedded webview facade: `NativeWebView`
- Dialog facade: `NativeWebDialog`
- Authentication facade: `WebAuthenticationBroker`
- Shared contracts and controller orchestration
- Platform adapters for Windows, macOS, Linux, iOS, Android, and Browser
