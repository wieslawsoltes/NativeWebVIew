# Platforms

Use the platform pages together with the [Platform Support Matrix](../reference/platform-support-matrix.md) when you need the honest current repo runtime status instead of the broader capability contract.

- `NativeWebView` control runtime paths now exist on Windows, macOS, Linux, iOS, Android, and Browser.
- `NativeWebDialog` is intentionally desktop-only in the current repo runtime and is implemented on Windows, macOS, and Linux.
- `WebAuthenticationBroker` now has runtime implementations on Windows, macOS, Linux, iOS, Android, and Browser.
